## Project Title: Sales Performance Diagnostic Dashboard

### Project Overview
This interactive dashboard analyzes sales pipeline data to identify low-margin regions, visualize sales trends, and support strategic decision-making. Built using Power BI, it provides insights into profitability, customer segmentation, and sales recovery strategies.

### Objectives
- Identify profit trends and sales opportunities
- Improve revenue performance through data-driven insights
- Provide management with an actionable dashboard

### Tools & Techniques
- **Power BI** (DAX, advanced visuals, drill-down reports)
- **Data Cleaning** (Power Query)
- **KPI tracking**
